Detailed Description

Contain tools for managging static and dynamic deadlocks.

Defintion:
Static DL: it is a square from which it is impossible to push a box on any goal (example: corners)
Dynamic DL: it is a node which will create an unsolvable situation becuause of boxes which blocks each other (example: square composed of 4 boxes)
